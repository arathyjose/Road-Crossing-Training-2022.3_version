# Road Crossing Training Simulator

> An interactive 3D simulation built in Unity to teach road safety and pedestrian awareness — practice crossing roads in a realistic virtual environment, with zero real-world risk.

## 🚦 The Problem

Teaching road safety — especially to children — is difficult in real-world settings. Busy traffic, unpredictable conditions, and the risk of accidents make hands-on practice dangerous. Traditional classroom methods lack the engagement needed to build real instincts.

## 💡 The Solution

**Road Crossing Training Simulator** puts the user inside a realistic 3D traffic environment where they can practice crossing roads at their own pace. Vehicles follow real signal logic, move dynamically, and respond to the environment — giving users a genuine sense of traffic flow without any danger.

---

## ✨ Features

- 🚗 **Dynamic vehicle movement** — cars follow realistic traffic patterns and speeds
- 🚦 **Working traffic signals** — lights cycle through red, yellow, and green with proper timing
- 🧍 **User-controlled avatar** — walk, stop, and cross using keyboard/controller input
- 🌍 **3D environment** — fully modelled road scene with lanes, pavements, and crossings
- 🔁 **Repeatable scenarios** — practice the same crossing as many times as needed
- 🛡️ **Zero real-world risk** — safe space to build pedestrian awareness and instincts

## 🚀 Getting Started

**Prerequisites:** Unity 2022.3 LTS installed via Unity Hub.

```bash
# Clone the repository
git clone https://github.com/arathyjose/Road-Crossing-Training-2022.3_version.git

# Open in Unity Hub
# Click "Add" → select the cloned folder → open with Unity 2022.3 LTS

# Press ▶ Play in the Unity Editor to run the simulation
```

## 🔄 How It Works

```
User launches simulation
        ↓
3D road environment loads with active traffic
        ↓
Traffic signals cycle — red / yellow / green
        ↓
Vehicles move dynamically across lanes
        ↓
User navigates to cross safely
        ↓
Practice repeats — no consequences, just learning
```
